At Least 4 Pages – Your website should contain a minimum of four distinct pages that align with a cohesive theme.
- **Homepage**
- **Menu**
- **About**
- **Contact**

Responsive Design – Ensure the website layout adjusts smoothly across different screen sizes (desktop, tablet, and mobile).
- **Page adjusts when scaling up from mobile to tablet, and tablet to desktop sizes.**
- **Responsivness adjusts mainly in the navigation header, but some font sizes change as well in the rest of the site.**

Media Queries – Use CSS media queries to make design adjustments at specific screen widths for optimal user experience on all devices.
- **Used media query to adjust navigation bar padding as site adjusts from mobile to tablet, and tablet to desktop sizes.**
- **About us paragraph font size adjusts to make text easier to see on larger screens.**
- **Entire site adjusts in size in an attempt to keep everything on the screen as much as possible to limit the need to scroll.**

Navigation Menu – Include a clear, user-friendly navigation menu to allow visitors to easily access each page of the site.
- **Navigation menu and dropdown menu are the same on the site.**
- **Navigation menu appears when hovering the active page heading, allowing access to the rest of the site's pages.**
